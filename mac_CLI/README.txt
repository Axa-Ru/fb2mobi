HOW TO USE

1. Copy fb2mobi.app to the place you like (e.g. /Applications)
2. Edit FB2MOBI_APP variable in "fb2mobi" and "synccovers" shell scripts included in this folder
   (If you have copied fb2mobi.app to /Applications, you don't have to edit)
3. Now "fb2mobi", "synccovers" shell script works as a command-line utilities.
